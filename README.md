# A-Multi-level-Causal-Inference-Method
We introduce a method that integrates causal inference with multi-level modeling (MCBNS) to develop a prediction model grounded in stable causal relationships and refine it at a fine-grained scale.

bnlearn_lme5_2020_shen.R, bnlearn_lme5_2010_shen.R, bnlearn_lme5_2000_shen.R: Multiscale correction
